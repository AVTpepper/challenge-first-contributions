Welcome! We are all collaborating on this single repository. Follow these steps carefully to add your name to the digital yearbook.

Step 1: Clone the Repository
Open your terminal/command prompt and run: git clone https://github.com/AVTpepper/challenge-first-contributions.git

Step 2: Create Your Own Branch
Important: Do not work on the main branch. Create a separate branch for your changes: git checkout -b add-profile-YOURNAME (Replace YOURNAME with your actual name, or github username)

Step 3: Create Your File
Open the repository folder in your code editor

Open the participants/ folder.
Create a new file named your-name.md.
Type a short bio or "Hello World!" inside the file and save it.

Step 4: Commit and Push
Run these three commands in your terminal:

Bash

git add .
git commit -m "Add profile for YOURNAME"
git push origin add-profile-YOURNAME

Step 5: Open the Pull Request (PR)
Go to the main page of this repository on GitHub.

You will see a yellow bar that says "Compare & pull request." Click it.

Title your PR: Add profile for [Your Name].

Click the green Create pull request button.
